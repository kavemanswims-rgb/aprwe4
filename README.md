# Western Payroll Mac - YTD Import Saved Fixed

This version saves the payroll database at:

~/Western_Payroll_Data/western_payroll.db

That means imported or manually entered year-to-date totals only need to be done one time. The app folder can be moved or replaced without losing the saved YTD data.

Run on Mac:

```bash
cd ~/Downloads
unzip western_payroll_mac_ytd_import_saved_fixed.zip
cd western_payroll_mac_ytd_import_saved_fixed
./run_mac.sh
```

Open http://localhost:8501
